# MRehber-Anti-K-f-r-Eklentisi
MRehber Gelişmiş Sohbet Moderasyon Eklentisi  Minecraft sunucunuz için Google Gemini 2.0 Flash yapay zeka destekli otomatik moderasyon sistemi. Eklenti, önce yerel filtre sistemi mesajları hızlıca tarar. Şüpheli durumlarda veya şifreli küfürlerde yapay zeka devreye girerek detaylı analiz yapar.
